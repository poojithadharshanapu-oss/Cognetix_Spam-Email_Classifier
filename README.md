# Spam Email Classifier

## Overview
Spam Email Classifier is a Machine Learning project that automatically classifies emails as Spam or Ham (Not Spam). The model uses Natural Language Processing (NLP) techniques and TF-IDF Vectorization to convert text data into numerical features before training a classification algorithm.

## Features
- Email text preprocessing
- Stopword removal using NLTK
- TF-IDF feature extraction
- Spam/Ham classification
- Performance evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
- Confusion Matrix Visualization
- Custom email prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn

## Project Workflow
1. Load and explore email dataset.
2. Clean and preprocess text data.
3. Remove stopwords and unwanted characters.
4. Convert text into numerical features using TF-IDF.
5. Split data into training and testing sets.
6. Train the classification model.
7. Evaluate model performance.
8. Predict whether a new email is Spam or Ham.

## Results
- Accuracy: 96.5%
- Precision: 100%
- Recall: 74.7%
- F1 Score: 85.5%
